# Paint App Using React

## This is the source-code for the paint app created with reactjs

### Features

    [] Free Hand Drawing
    [] Eraser
    [] Rectangle Drawing - with cordinates displayed
    [] Clear Canvas


### To install the app

    [] git clone 'https://github.com/anixsam/paint-reactjs.git
    [] cd paint-reactjs
    [] npm i
    [] npm start
